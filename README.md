# Socket Programming Tool in Python

This tool implements various socket programming functionalities including:

- **Port Scanner**: Scans a target IP for open ports.
- **Banner Grabber**: Retrieves banner messages from a given IP and port.
- **TCP Client/Server**: Enables two-way communication over TCP.
- **UDP Client/Server**: Enables two-way communication over UDP.
- **Chat Application**: Multi-client chat with basic chat commands.

## Installation

Ensure you have Python 3 installed. Clone the repository and install dependencies:

```bash
git clone https://github.com/kaviyarasank2004/Socket_Programming_Tool
cd Socket_Tool
